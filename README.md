# goodwe-connector
A python library to connect with Goodwe API Rest and get production data about solar panels.

## Goodwe connector user configurarion.

- System Id
- Account
- Password

Example of goodwe_config.json

```
{
    "goodwe_api_connection":{
        "system_id":"",
        "account":"",
        "password":""
    }
}
```
